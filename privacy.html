Privacidad

CONSIA es privacy-first. La memoria y el almacenamiento de información se activan solo cuando el usuario lo elige (opt-in).

Qué datos usamos: los mínimos necesarios para operar el servicio (por ejemplo, sesión, país/idioma, métricas agregadas).

Memoria: si el usuario la activa, CONSIA recuerda señales útiles para mejorar la experiencia. El usuario puede desactivarla.

Seguridad: aplicamos controles de acceso y medidas razonables para proteger la información.

Contacto: privacy@consia.world
